## Usage

* `git clone https://github.com/moldcraft/color-scheme.git`
* `cd color-scheme`
* `mkdir -p ~/.YOUR_IDE/config/colors`
* `ln -s $(pwd)/moldcraft.icls ~/.YOUR_IDE/config/colors/`
