## Rules

Component | Color
---|---
Background | dark gray
Keywords | orange
Comments | violet
Strings | green
Global Variable | dark blue
Local Variable | blue
Argument Variable | light blue
Functions/Methods | dark yellow
Caret | `#00FF00`
TODO/FIXME | `#FF0000`
